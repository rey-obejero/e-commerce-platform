# E-Commerce Platform

## Project Overview

E-Commerce Platform is a single-merchant digital commerce solution.

## Prerequisites

This application was built with Express.js, TypeScript, React, JavaScript, and PostgreSQL. To run or develop, the following tools are required:

- Node.js
- NPM

## Getting Started

Before proceeding, ensure the necessary tools are installed.

### Web Client

1. Install the necessary dependencies with:

    ```console
    npm install
    ```

2. Start the web client with:

    ```console
    npm start
    ```

3. Access the application via [http://localhost:3000](http://localhost:3000).

### API Server

1. Install the necessary dependencies with:

    ```console
    npm install
    ```

2. Define the server hostname and port in an ENV file. Refer to the [ENV schema](/server/src/config/env.ts) for the valid values.

3. Start the web client with:

    ```console
    npm run start:dev
    ```
